# Exercício 02 — Branches

Foram criadas múltiplas branches para desenvolvimento isolado de funcionalidades.

## Branches utilizadas

- feat/calculadora
- feat/historico-operacoes
- docs

## Estratégia

Cada funcionalidade foi desenvolvida em uma branch separada e integrada à branch main via Pull Request.

## Pull Requests

Os Pull Requests foram utilizados para revisar e descrever as alterações antes do merge.

Essa abordagem permite melhor organização e rastreabilidade do desenvolvimento.
