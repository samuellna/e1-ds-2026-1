# Portfólio Git — Calculadora

Este repositório foi desenvolvido como parte da disciplina de Desenvolvimento de Software (CIn-UFPE), com o objetivo de demonstrar o uso prático do Git.

## Objetivos

- Utilizar commits semânticos (Conventional Commits)
- Trabalhar com branches e merges
- Utilizar Pull Requests como unidade de revisão
- Resolver conflitos manualmente
- Documentar o processo de aprendizagem

## Estrutura do Repositório

## Projeto

O projeto consiste em uma calculadora em Python com:

- Operações básicas (+, -, \*, /)
- Cálculo de Bhaskara
- Histórico de operações salvo em JSON

## Como executar

`python nome_do_arquivo.py`

## Uso de Git

Durante o desenvolvimento, foram utilizados:

Branches para organização de funcionalidades
Pull Requests para integração
Commits semânticos para rastreabilidade
